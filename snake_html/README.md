# HTML 版贪吃蛇

这是一个不需要 Python 的网页版贪吃蛇小游戏，直接用浏览器打开 `index.html` 即可运行。

## 运行方法

双击打开：

```text
snake_html/index.html
```

也可以在浏览器地址栏打开该文件。

## 操作方式

- 方向键或 W、A、S、D：控制蛇移动
- P 或空格：暂停/继续
- Enter：在主菜单开始游戏
- 鼠标：点击菜单按钮

## 功能

- 主菜单、难度选择、暂停和结算界面
- 三种难度：简单、普通、困难
- 普通食物加 10 分
- 每吃 5 个普通食物出现特殊食物
- 特殊食物限时 5 秒，快消失时闪烁，吃到加 50 分
- 撞墙或撞到自己后游戏结束
- 最高分使用浏览器 `localStorage` 自动保存
- 使用 Web Audio 生成简单音效，不需要额外音频文件
